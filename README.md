# MorseCodeApplication
MorseCodeApplication is a simple application which translates text written in the Latin alphabet into its Morse code representation as well as the other way around. The code can then be played in the audio format, as a series of short and longer sounds.

The graphic user interface contains fields for the text and the Morse code, a playback button, a set of buttons provided for entering the code, and a settings panel.

The minimized version of the window also provides a dragging button – ‘✣’, on the left, which makes it possible to change the window’s location on the screen. The two buttons on the right allow the user to switch between the settings menu and the main panel containing the text fields.

The input text gets automatically converted to capital letters, as the application does not differentiate between lower and uppercase. 
The letters in the Morse code transcript are represented as a series of dots and dashes. A space separates individual letters and a forward slash separates words.

The user can also enter a Morse code value to be translated into text. Clicking the value button inserts a dot, while holding it inserts a dot. Similarly with the space button, a click separates the letters, while a longer press separates the words. In case of a mistake made, a backspace button is also provided.
